# AfterBuy Docs

Static public policy and support pages for the AfterBuy iOS app.

The site uses the same bright, rounded AfterBuy design language as the iOS app and the production `AfterBuyLogoMark` asset. Shared styling lives in `styles.css`, and web-ready logo assets live in `assets/`.

These pages are intended for App Store Connect fields:

- Privacy Policy URL: `https://marcusperdue.github.io/AfterBuy-Docs/privacy.html`
- Support URL: `https://marcusperdue.github.io/AfterBuy-Docs/support.html`
- User Privacy Choices URL: `https://marcusperdue.github.io/AfterBuy-Docs/privacy-choices.html`
- Terms URL: `https://marcusperdue.github.io/AfterBuy-Docs/terms.html`

GitHub Pages should be served from the `main` branch root.

## Local preview

From this directory, run:

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.
